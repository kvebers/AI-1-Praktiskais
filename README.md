# AI-1-Praktiskais

## Branch Rules

| Branch | Required Approvals |
| ------ | ------------------ |
| `main` | 2 approvals        |
| `dev`  | 1 approval         |

## Git Workflow

```
feature/** → dev → main
```

1. Create a feature branch: `feature/your-feature-name`
2. Open a pull request into `dev` from `feature/your-feature-name`
3. Open a pull request from `dev` to `main`
